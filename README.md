# Social Linker 

[![Build Status](https://dev.azure.com/shayneboyer/social-linker/_apis/build/status/spboyer.social-linker?branchName=master)](https://dev.azure.com/shayneboyer/social-linker/_build/latest?definitionId=2&branchName=master)

This tool is just a simple VueJS application to add `/?WT.mc_id=event-channel-alias` path to Microsoft based links.

This is used by the Cloud Advocates to help track Microsoft-based traffic.
